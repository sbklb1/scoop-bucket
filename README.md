# Scoop Bucket

A personal, unofficial [scoop](https://scoop.sh) bucket of miscellaneous Windows applications.

## Usage

```powershell
scoop bucket add sbklb1 "https://github.com/sbklb1/scoop-bucket"
scoop install sbklb1/<app_name>
```

## How do I contribute new manifests?

To make a new manifest contribution, please read the [Contributing
Guide](https://github.com/ScoopInstaller/.github/blob/main/.github/CONTRIBUTING.md)
and [App Manifests](https://github.com/ScoopInstaller/Scoop/wiki/App-Manifests)
wiki page.

## Publish to scoop.sh

To share your bucket, all you need to do is tell people how to add your bucket, i.e. by
running the command in step 3. If you want your bucket listed in the Scoop Directory,
add a topic of scoop-bucket to its github page.
